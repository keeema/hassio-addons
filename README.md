# Keeema's Home Assistant Add-ons

Custom add-ons repository for Home Assistant.

## Installation

1. Open Home Assistant
2. Go to **Settings** > **Add-ons** > **Add-on Store**
3. Click the three dots in the top right corner and select **Repositories**
4. Add this URL: `https://github.com/keeema/hassio-addons`
5. Click **Add** and then **Close**
6. The add-ons from this repository will now appear in the store
